# UIDAI Data Hackathon 2026

## Project Title
A Policy-Oriented Analysis of Aadhaar Enrolment and Update Trends in India

## Description
This repository contains the Google Colab notebook used for analyzing Aadhaar enrolment,
demographic update, and biometric update datasets provided by UIDAI as part of the
UIDAI Data Hackathon 2026.

The analysis focuses on identifying enrolment imbalances, lifecycle-driven update patterns,
geographic disparities, and temporal trends to support policy-oriented decision-making.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Note
All datasets used are aggregated and anonymized UIDAI datasets.
No personally identifiable information (PII) is used.
